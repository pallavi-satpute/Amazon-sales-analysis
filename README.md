# Amazon Sales Analysis

This project analyzes Amazon sales data to extract meaningful business insights using Python and data visualization techniques.

## Objectives
- Understand sales trends
- Identify top-performing products
- Analyze revenue patterns

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Monthly sales trends analysis
- Category-wise performance
- Revenue distribution
